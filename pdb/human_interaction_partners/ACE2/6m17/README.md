# PDB 6m17

## Overview

**Protein name**: human interaction partners

**Organism**: ACE2

**Method**: Cryo-EM

## Description

Receptor binding domain in complex with full lenght ACE2 and amino acid transporter B0AT1. Study highlight the entry mechanism of SARS-CoV-2 in cells.
