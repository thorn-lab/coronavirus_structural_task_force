# PDB 3atw

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV forming a dimer with a peptide like enzyme inhibitor in the active site. This X-ray crystal structure was phased using PDB entry 2ZU5 for molecular replacement & has data to a resolution of 2.36Å.

## Basefolder

3atw.pdb and 3atw.cif - the coordinates of the threedimensional molecular model

3atw-sf.cif - deposited diffraction data for this PDB entry

3atw.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
