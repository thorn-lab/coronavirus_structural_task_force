# PDB 5rgz

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV-2 forming a dimer. This protein is bound to a small molecule frag, 2-(3-cyanophenyl)-N-(pyridin-3-yl)acetamide. This X-ray crystal structure has data to a resolution of 1.52Å

## Basefolder

5rgz.pdb and 5rgz.cif - the coordinates of the threedimensional molecular model

5rgz-sf.cif - deposited diffraction data for this PDB entry

5rgz.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
