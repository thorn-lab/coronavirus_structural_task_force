# PDB 6xbg

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV with the peptide like enzyme inhibitor UAW246 in the active site. This X-ray crystal structure was phased using PDB entry 7brr for molecular replacement & has data to a resolution of 1.45Å.

## Basefolder

6xbg.pdb and 6xbg.cif - the coordinates of the threedimensional molecular model

6xbg-sf.cif - deposited diffraction data for this PDB entry

6xbg.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
