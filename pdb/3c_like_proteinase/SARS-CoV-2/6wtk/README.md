# PDB 6wtk

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV with a peptide like enzyme inhibitor in the active site. This X-ray crystal structure was phased using PDB entry 6y7m for molecular replacement & has data to a resolution of 2Å.

## Basefolder

6wtk.pdb and 6wtk.cif - the coordinates of the threedimensional molecular model

6wtk-sf.cif - deposited diffraction data for this PDB entry

6wtk.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
