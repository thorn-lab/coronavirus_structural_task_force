# PDB 6xhu

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV. This data was collected at room temperature. This X-ray crystal structure was phased using PDB entry 6wqf for molecular replacement & has data to a resolution of 1.8Å.

## Basefolder

6xhu.pdb and 6xhu.cif - the coordinates of the threedimensional molecular model

6xhu-sf.cif - deposited diffraction data for this PDB entry

6xhu.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
