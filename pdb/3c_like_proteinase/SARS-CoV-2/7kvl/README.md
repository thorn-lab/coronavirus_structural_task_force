# PDB 7kvl

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

X-ray diffraction crystal structure of SARS-CoV-2 main protease immature form. Main protease is responsible for cutting the polyprotein into functional parts.

## Basefolder

7kvl.pdb and 7kvl.cif - the coordinates of the threedimensional molecular model

7kvl-sf.cif - deposited diffraction data for this PDB entry

7kvl.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
