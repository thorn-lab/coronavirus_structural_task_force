# PDB 5rgx

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV-2 forming a dimer. This protein is bound to a small molecule frag, 2-(3-cyanophenyl)-N-(4-methylpyridin-3-yl)acetamide. This X-ray crystal structure has data to a resolution of 1.69Å

## Basefolder

5rgx.pdb and 5rgx.cif - the coordinates of the threedimensional molecular model

5rgx-sf.cif - deposited diffraction data for this PDB entry

5rgx.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
