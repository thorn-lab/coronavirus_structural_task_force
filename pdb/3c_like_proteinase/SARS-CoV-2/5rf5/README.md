# PDB 5rf5

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV-2 forming a dimer. This protein is bound to a small molecule frag, 1,1-bis(oxidanylidene)thietan-3-ol. This X-ray crystal structure has data to a resolution of 1.74Å

## Basefolder

5rf5.pdb and 5rf5.cif - the coordinates of the threedimensional molecular model

5rf5-sf.cif - deposited diffraction data for this PDB entry

5rf5.mtz - structure factors after automatic refinement of molecular model against the diffraction data
