# PDB 6m0k

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV with a Fluorine based ligand, 11b, in the active site. This X-ray crystal structure was phased using PDB entry 6lu7 for molecular replacement & has data to a resolution of 1.7Å.

## Basefolder

6m0k.pdb and 6m0k.cif - the coordinates of the threedimensional molecular model

6m0k-sf.cif - deposited diffraction data for this PDB entry

6m0k.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
