# PDB 6lu7

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from SARS-CoV with the peptide like enzyme inhibitor N3 in the active site. This X-ray crystal structure was phased using PDB entry 2hob for molecular replacement & has data to a resolution of 2.16Å.

## Basefolder

6lu7.pdb and 6lu7.cif - the coordinates of the threedimensional molecular model

6lu7-sf.cif - deposited diffraction data for this PDB entry

6lu7.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
