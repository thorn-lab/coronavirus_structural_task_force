# PDB 7ali

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Full length 3C-like protease from Sars-CoV-2 with 1.65A resolution. The spacegroup is P2(1).
