# PDB 6zrt

## Overview

**Protein name**: 3c like proteinase

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of 3C-like protease from SARS CoV2 (Homodimer) in complex with inhibitor Telaprevir. Docking studies, enzyme inhibition and co-crystal structure analyses showed that telaprevir does not inhibit.

## Basefolder

6zrt.pdb and 6zrt.cif - the coordinates of the threedimensional molecular model

6zrt-sf.cif - deposited diffraction data for this PDB entry

6zrt.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
