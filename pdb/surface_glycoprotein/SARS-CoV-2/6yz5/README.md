# PDB 6yz5

## Overview

**Protein name**: surface glycoprotein

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

Crystal structure of the SARS-CoV-2 spike RBD in complex with the nanobody H11-D4. Non polymer only dimer, spike protein with one chain and nanobody with one chain. 

## Basefolder

6yz5.pdb and 6yz5.cif - the coordinates of the threedimensional molecular model

6yz5-sf.cif - deposited diffraction data for this PDB entry

6yz5.mtz - structure factors after automatic refinement of molecular model against the diffraction data.
