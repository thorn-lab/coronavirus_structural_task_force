**Organism**: SARS-CoV-2

## Description

Part of PanDDA analysis group deposition. SARS-CoV-2 Nsp3 macrodomain crystallographic fragment screen. The potential binding ligand(s): [RZS](https://www.rcsb.org/ligand/RZS).

5s3k.pdb and 5s3k.cif - the coordinates of the threedimensional molecular model

5s3k-sf.cif - deposited diffraction data for this PDB entry
