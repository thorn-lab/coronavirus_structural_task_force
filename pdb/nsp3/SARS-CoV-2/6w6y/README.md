## Overview

**Protein name**: nsp3

**Organism**: SARS-CoV-2

**Method**: X-Ray Diffraction

## Description

1,5 Å Crystal Structure of Mac1 domain of NSP3 from SARS CoV-2 in complex with AMP. This domain has a function as an ADP ribose 1"-phosphatase."
