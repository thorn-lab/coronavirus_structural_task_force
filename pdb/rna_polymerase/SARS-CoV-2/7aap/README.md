# PDB 7aap

## Overview

**Protein name**: rna polymerase

**Organism**: SARS-CoV-2

**Method**: Cryo-EM

## Description

Electron microscopy structure of SARS-CoV2 RNA-dependent RNA polymerase (Rdrp with subunits Nsp7-Nsp8-Nsp12) in complex with template:primer dsRNA and favipiravir-RTP.
